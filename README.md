# Maria Fernanda Makeup Artist

Landing page responsiva para uma profissional de maquiagem, com visual luxuoso, tema escuro, chamadas para agendamento via WhatsApp e secoes interativas para apresentar trabalhos, servicos e depoimentos.

## Link publicado

https://clebervieira25.github.io/Maria-Fernanda/

## Recursos

- Hero em tela cheia com foto da profissional
- Navegacao fixa com links internos
- Botoes de agendamento pelo WhatsApp
- Icones das redes sociais com cores da marca
- Portfolio com filtros por categoria
- Secao sobre a profissional
- Cards de servicos
- Comparacao visual interativa
- Depoimentos navegaveis
- FAQ expansivel
- Background animado com estetica de beleza e luxo
- Layout responsivo para desktop, tablet e celular

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Personalizacao

Para trocar o numero do WhatsApp, substitua `5599999999999` nos links do arquivo `index.html`.

Para trocar a foto principal, substitua o arquivo `mariafernanda.jpg` ou altere o `src` da imagem com a classe `hero-photo`.

As imagens adicionais podem ser organizadas dentro da pasta `images/`.
